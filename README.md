# E-Commerce Sales Analysis 

## Project Overview
Exploratory data analysis of a Brazilian e-commerce 
dataset from Olist stores containing ~100k orders 
from 2016 to 2018.

## Objectives
- Understand customer behaviour and loyalty
- Analyze order and delivery performance
- Identify geographic trends across Brazilian states
- Provide data-driven business recommendations

## Key Findings
- 97.02% of orders were successfully delivered
- Only 3.36% of customers are repeat buyers
- August is the busiest month for orders
- Monday is the peak shopping day
- SP, RJ and MG account for 66.61% of all orders
- Northern states wait up to 29 days for delivery

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset
Download the dataset from Kaggle:
[Brazilian E-Commerce Dataset](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce)

## How to Run
1. Clone this repository
2. Download the dataset from Kaggle
3. Place CSV files in an `archive` folder
4. Open `Sales_Analysis_Project.ipynb` in Jupyter Notebook
5. Run all cells
